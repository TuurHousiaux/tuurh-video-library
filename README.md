# tuurh-video-library

Whitelist van video libraries / playlists op `https://tuurh.be/me/videos/`.

Beheer: bewerk `src/libraries.json`. Items met `"draft": true` worden niet getoond.

Optioneel veld `"image"`: pad of URL naar een thumbnail (bijv. `images/mijn-cover.png` in `src/images/`). Zonder image krijgt de card een cover gegenereerd uit de titel.
